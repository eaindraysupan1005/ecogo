# **EcoGo App**

EcoGo is a mobile app designed to encourage eco-friendly habits and promote sustainability. Users can select habits, log their completion, track their progress through levels, and view leaderboards. Built using **React Native**, **Firebase**, and **Java**, this app aims to inspire positive environmental change.

---

## **Features**

### **Core Functions**
1. **Habit Selection and Completion**  
   - Choose eco-friendly habits (e.g., reducing water use, biking to work).  
   - Log daily habit completion to earn points.

2. **Points and Levels System**  
   - Earn points based on the impact of completed habits.  
   - Advance through levels (Wood → Iron → Bronze → Diamond).  
   - Dynamic progress bar and badges for milestones.

3. **Leaderboard**  
   - View all-time rankings.  
   - Highlights top-ranking users and shows individual rank.

4. **Weekly History View**  
   - Review completed habits and points earned over the past 7 days.  
   - Visualize progress with weekly charts.

5. **Profile Management**  
   - Edit personal details (e.g., avatar, username, email).  
   - Display current level, points, badges, and rank.

6. **Notifications and Reminders**  
   - Get alerts for milestone achievements and feedback on habits.  

7. **User Insights and Analytics**  
   - View charts tracking weekly points for up to 5 weeks.  
   - Insights into total habits completed.

---

## **Tech Stack**
### **Frontend**
- **React Native**: For building the cross-platform user interface.

### **Backend**
- **Firebase**:
  - Authentication for login and sign-up.
  - Firestore for storing user data, points, and habits.
  - Cloud Functions for server-side logic.
  - Firebase Cloud Messaging for notifications.

### **Native Features**
- **Java** (Android): For custom native modules and performance optimization.
